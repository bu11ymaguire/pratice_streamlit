# pratice_streamlit

Final Presentation
불균형 데이터셋에 대해 앙상블 모델의 성능을 개선하고자 SMOTE-NC(Synthetic Minority Over-sampling Technique for Nominal and Continuous)를 적용했으나, 과적합(overfitting)이나 합성 샘플 생성 과정에서의 노이즈로 인해 예상과 달리 성능이 저하되었습니다. 그러나 클래스 불균형 문제를 인지하고 이를 해결하기 위한 체계적인 접근 방식과 시도 자체를 높이 평가받아 2등이라는 좋은 성과를 거두었습니다.
